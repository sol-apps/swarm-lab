API HELP 
agent      the one being updated. x, y = position (px); vx, vy = velocity (px/s —
           the engine integrates position and handles edges AFTER your code runs);
           species (int, picks the colour); phase, state, energy, glow,
           justFlashed; mem = free scratch object, all yours.

neighbors  agents within the preset's perception radius, self excluded.
           Read them freely; mutating them is cheating (the predator does it anyway).

world      width, height, dt, time, frame, edgeMode ('wrap'|'bounce'),
           rng() = seeded random 0..1,
           wrapDx(fromX, toX), wrapDy(fromY, toY) = shortest delta across wrapped edges,
           pheromone (ANTS only): deposit(x, y, amt, ch), sense(x, y, ch) — ch 0 or 1,
           food = [{x, y, r}], nest = {x, y, r} (ANTS only).

params     current slider values by key (see PARAMETERS panel).
           PARTICLE LIFE adds params.matrix[rowSpecies][colSpecies] = -1..+1.

helpers    dist(ax, ay, bx, by)   clamp(v, lo, hi)   lerp(a, b, t)
           angleTo(ax, ay, bx, by)   limit(vx, vy, max) → [vx, vy]

notes      APPLY compiles; syntax errors keep the old code running.
           A runtime error or code slower than ~50ms/tick pauses the sim — fix
           and APPLY, or REVERT to the preset original. An infinite loop
           (while(true)) will still hang the tab: the watchdog can only run
           between ticks. Nothing you type here leaves your browser unless
           you press SHARE.
